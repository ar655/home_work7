вопрос: что такое Hash Map?

ответ: Hash Map это структура данных котороя хранит
пары ключ-значения



вопрос: что такое Linked List?

ответ: linked list - это структура данных,
в которой несколько значений хранятся линейно.
Каждое значение содержит своё собственное значение узла,
а также содержит данные вместе со ссылкой на следующий узел в списке


вопрос : в каких случаях использовать Linked List?

ответ: linked list используется когда есть активная работа
(вставка/ удаление ) c серединой списка


вопрос: в каких случаях использовать  Hash Map?

ответ: hashmap  используется для быстрого поиска,вставления,удаления,и
для эффективного хранения большого количества пар ключ-значение в словаре
